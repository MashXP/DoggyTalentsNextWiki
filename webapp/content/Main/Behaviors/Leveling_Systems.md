---
title: Leveling Systems
---

Dogs level up when being given [Treats](Items#Treats). Each treat can level a dog up to a certain level until it reaches the maximum &quot;normal&quot; level of 60, then the maximum &quot;kami&quot; level of 30 (in a total of 90).

The **maximum** amount of Talent points you can get is **105** (15+20+20+20+30). *(Or it can be bypassed with the [Creative Collar](Creative_Collar))*.

## Normal leveling
The normal leveling system is the default leveling path, in which the dog just gains more Talent Points as well as an additional **<code>1 HP</code>** with every level. Once it reaches the maximum level for this path, it has total health of **<code>27HP</code>**.

| Treat | Description |
| --- | --- |
| ![Normal Leveling](images/normal.png "Normal Leveling") | [Training Treat](Training_Treat) can level the dog up to level **20**. |
| ![Super](images/super.png "Super") | [Super Treat](Super_Treat) can level the dog up to level **40**. |
| ![Master](images/master.png "Master") | [Master Treat](Master_Treat) can level the dog up to level **60**, which is the **maximum** level for the normal leveling system. |

## Kami Leveling

| Icon | Description |
| --- | --- |
| ![Kami Leveling](images/kami.png "Kami Leveling") | The kami leveling system goes beyond the normal leveling path, in which the dog just gains more Talent Points as well as additional health with each level. Once it reaches the maximum level for this path, it will not only have a total health of 30 HP, it will begin to give out Purple Particles *(this can be configured, default is false)*. |

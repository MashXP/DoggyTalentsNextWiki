**Dog Animation Debug Stick** is an item that grants direct manipulation and fine-tuning of dog animation behaviors.

## Obtaining
Press **<code>E</code>** in Creative mode to access the Creative menu.

## Usage

- To configure the Debug Stick, **Sneak + Right Click** while the item is in your hand and you are not targeting a dog. This will open the configuration menu, allowing you to switch between the three modes.

### **Modes**

#### **Set/Clear**
- **Function:** This mode allows you to bind a specific animation to the Debug Stick and toggle the dog's Animation Debug Mode.
- **Usage:**
  - First, select the desired animation within the configuration menu.
  - **Right Click** a dog with the Debug Stick in your main hand to:
    - Enter Animation Debug Mode if the dog is not already in it.
    - Play or Pause the bound animation.
  - <code>**Sneak + Right Click**</code> or <code>**Right Click**</code> with the "NONE" animation selected to make the dog exit Animation Debug Mode.

#### **Fine Tune**
- **Function:** This mode allows you to precisely adjust the frozen timestamp of an animation within Animation Debug Mode.
- **Usage:**
  - Ensure the targeted dog is in Animation Debug Mode.
  - **Right Click** the dog to advance the animation's frozen timestamp forward.
  - **<code>Sneak + Right Click</code>** the dog to rewind the animation's frozen timestamp backward. This allows you to examine specific frames or moments in the animation.

#### **YRot Lock**
- **Function:** This mode locks the dog's Y-axis rotation (yaw) to match the player's viewing direction.
- **Usage:**
  - Ensure the targeted dog is in Animation Debug Mode.
  - **Right Click** the dog to instantly align its facing direction with your own. This is useful for precise positioning and observation of animations from specific angles.
